# **App Name**: BOQ Tracker

## Core Features:

- BOQ Table View: Display BOQ items in a structured table format (description, unit, quantity, rate, amount).
- Automatic Calculation: Automatically calculate line item totals and the grand project total.
- BOQ Entry Management: Enable users to add, edit, and delete BOQ entries.
- Interactive Map: Display an interactive map showing drainage clearing locations with points indicating work areas and images.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a clean, professional look.
- Background color: White (#FFFFFF) to ensure readability and a clean aesthetic.
- Accent color: Darker blue (#4682B4) to highlight key elements and calls to action.
- Body and headline font: 'Inter', a sans-serif for clean, readable text throughout the application.
- Use a grid-based layout to organize content in a clear, logical manner.
- Incorporate simple, professional icons to represent different BOQ items and actions.
